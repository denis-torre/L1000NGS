Reusing Notebooks
All notebooks generated using the L1000 Notebook Generator can be downloaded and rerun on your local machine using a Jupyter Server. To rerun a notebook, follow the steps below:
  1. Install the Python module requirements (also available as a requirements.txt file):

  2. Download the desired Jupyter notebook from the L1000 notebook generator web application. An example notebook is provided above.
  3. Launch the Jupyter Server and navigate to the notebook.
